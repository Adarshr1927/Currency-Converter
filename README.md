# Currency Converter

A responsive and user-friendly Currency Converter web application built using HTML, CSS, and JavaScript. 
This project uses Fetch API to get real-time exchange rates and convert currencies instantly.

# Features
Convert between multiple international currencies.
Real-time exchange rate updates using API integration.
Interactive and clean user interface.
Country flags support for selected currencies.
Fast and responsive design.
Error handling for invalid API responses.

# Technologies Used
HTML5 – Structure of the application.
CSS3 – Styling and responsive design.
JavaScript – Functionality and DOM manipulation.
Fetch API – Fetching live exchange rate data from the API.

# How It Works
User selects the source and target currencies.
Enters the amount to convert.
Application fetches real-time exchange rates using Fetch API.
Converted amount is displayed instantly.

# Purpose
This project was created to practice:
API integration using Fetch API.
JavaScript asynchronous programming.
DOM manipulation
Building responsive frontend applications.
